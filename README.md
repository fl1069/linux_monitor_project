# linux_monitor_project

System Monitor
Monitor Linux system metrics and generate reports.

Requiremnts:
python3
psutil
pathlib


How to use?
pip install psutil
cd reporting
python3 rep_gen.py


Features
CPU/Memory/Disk usage
Process monitoring
Directory change tracking

Output
Report: report.txt
